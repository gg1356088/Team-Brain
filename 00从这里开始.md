# IncepVision Law — 团队大脑

> 这是 IncepVision Law 的团队知识库。这里放团队可以阅读和协作使用的人事、行政、自媒体和通用流程资料。
> 财务、税务、工资明细、客户账务、银行资料、客户原始文件、凭据、token、密钥、完整开发包不放在团队默认可见区。

---

## 先看这里

| 区域 | 用途 | 默认权限 |
|------|------|----------|
| 👥 人事模块 | 简历自动抓取、面试流程、笔试管理、HR 状态汇报 | 团队可读，负责人维护 |
| 🗂️ 行政模块 | 客户邮件整理、资料归档、Chat 案件存档、资产交接 | 团队可读，负责人维护 |
| 📣 自媒体模块 | 选题、发布流程、素材规范、账号运营 SOP | 自媒体负责人可写，团队可读 |
| 💼 财务模块 | 财务、税务、账务流程资料预留区 | 暂不开放，仅 admin |
| 90_Inbox_Team_Submissions | 团队临时提交资料、草稿、待整理内容 | 指定成员可写 |
| ⚙️ 系统支撑 | Agent 规则、API 结构、内部系统说明 | 管理/技术人员限定 |
| 📦 开发者完整版 | 原始开发资料和完整导出包 | 管理/技术人员限定 |
| 99_Admin_Control | 权限规则、管理员控制资料 | 管理员限定 |

## 使用规则

1. 正式 SOP、政策、模板由负责人维护，普通成员不要直接改正式区。
2. 新资料先放到 `90_Inbox_Team_Submissions`，由负责人整理后再进入正式模块。
3. 财务模块已预留，但暂不开放；财务、税务、工资、银行、客户账务、客户原始资料仍不得放入团队可见区。
4. 不上传凭据、token、密钥或完整开发包到团队可见区。
5. 需要开发者资料时，向管理员申请限定访问，不从团队默认入口散发。
